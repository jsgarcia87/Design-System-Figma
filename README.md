# FigmaTokens Pro 🚀

**FigmaTokens Pro** is an ultra-premium Design System Generator that bridges the gap between design and code. It extracts design tokens (colors, typography, and components) directly from Figma files and provides production-ready code and documentation.

## 🌟 Features

- **Automated Extraction**: Fetches colors, typography, and components via Figma REST API.
- **SaaS Pro Dashboard**: Beautiful, dark-mode interface with real-time search and filtering.
- **Durable Exports**:
  - CSS Variables (`:root`)
  - JSON Tokens (Standard format)
  - **design.md**: Markdown documentation.
  - **design.pdf**: Portable PDF documentation.
- **Figma Integration**: "Copy for Figma" feature to quickly import tokens into Figma plugins.
- **Demo Mode**: Test the application instantly with pre-extracted design data.

## 🛠️ Tech Stack

- **React + Vite**
- **Vanilla CSS** (Custom Design System)
- **Framer Motion** (Animations)
- **jsPDF** (PDF Generation)
- **Lucide-React** (Iconography)
